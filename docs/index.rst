epx-pop
===========

`epx-pop`_ is an internal Epistemix Python package containing tools for working
with `FRED`_ compatible synthetic populations.

.. _epx-results: https://github.com/Epistemix-com/epx-pop
.. _FRED: https://github.com/Epistemix-com/FRED-dev

The core functionality of the package includes:

- loading synthetic population data into Pandas DataFrames
